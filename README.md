# 🏋️ Fitness Bot — реальная оплата через Telegram Stars

## Почему Telegram Stars?

| | Telegram Stars | Click / Payme |
|---|---|---|
| Регистрация | Не нужна | Нужно ИП/ООО |
| Время настройки | 5 минут | 1-2 недели |
| Работает без сервера | ✅ Да | ❌ Нужен webhook |
| Страны | Весь мир | Только Узбекистан |
| Курс | ~$0.013 за Star | Сум |

---

## 🚀 Запуск за 5 минут

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Создать бота у @BotFather
```
/newbot
→ задать имя и username
→ скопировать токен
```

### 3. Настроить .env
```bash
cp .env.example .env
# Вставить BOT_TOKEN
```

### 4. Запустить!
```bash
python bot/bot.py
```

Всё. Больше ничего не нужно.

---

## 💰 Как получить деньги

Stars конвертируются в деньги через @BotFather:

```
@BotFather → /mybots → выбрать бота
→ Bot Payments → Withdraw Stars
```

Минимальная сумма для вывода: **1000 Stars** (~$13)

## 🔄 Как работает оплата

```
Пользователь вводит данные (возраст/рост/вес/цель)
        ↓
Бот создаёт заказ в SQLite (статус: pending)
        ↓
Бот отправляет инвойс (окно оплаты прямо в Telegram)
        ↓
Пользователь нажимает "Оплатить N Stars"
        ↓
Telegram списывает Stars и вызывает pre_checkout_query
        ↓
Бот отвечает ok=True
        ↓
Telegram вызывает successful_payment — Stars списаны!
        ↓
Бот обновляет статус заказа → paid
        ↓
Бот генерирует и отправляет программу тренировок 🎉
```

Никакого webhook, ngrok или внешнего сервера!
